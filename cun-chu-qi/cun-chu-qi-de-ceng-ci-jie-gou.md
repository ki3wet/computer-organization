# 存储器的层次结构

## 存储器的三个特性关系

![](../.gitbook/assets/image%20%2838%29.png)

* 价格：由高到低
* 容量：由小到大
* 速度：由快到慢
* 软件、硬件或软硬件结合，组成存储体系
* 一二三级缓存在CPU中

## 缓存—主存层次和主存—辅存层次

![](../.gitbook/assets/image%20%2845%29.png)

|  缓存—主存层次 | 主存—辅存层次 |  |
| :--- | :--- | :--- |
| 连接方式 | 硬件方法连接 | 软硬件结合方法连接 |
| 应用地址 | 实地址（物理地址） | 虚地址 |
| 解决的问题 | CPU与主存的速度不匹配 | 主存容量小问题 |
