# 动态RAM与静态RAM比较

## 

| 项目 | DRAM（主存） | SRAM（缓存） |
| :--- | :--- | :--- |
| 存储原理 | 电容 | 触发器 |
| 集成度 | 高（一个晶体管一个电熔） | 低（六个晶体管） |
| 芯片引脚 | 少 | 多（高速度） |
| 功耗 | 少（刷新用电） | 高 |
| 价格 | 低 | 高 |
| 速度 | 慢（主要有读写与充电） | 快 |
| 刷新 | 有 | 无 |
